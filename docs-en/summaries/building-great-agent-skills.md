# How to build great agent skills: a 4-point checklist

> Skills proliferated; a good-vs-bad rubric did not. Matt Pocock distills it to four checks: **Trigger, Structure, Steering, Pruning**.

A short executive summary of Matt Pocock's talk. Full checklist below.

## TL;DR

| # | Point | What it answers |
|---|-------|-----------------|
| 1 | **Trigger** | How is the skill invoked — by a human or the model? |
| 2 | **Structure** | What is the skill built from (steps + reference)? |
| 3 | **Steering** | How do you make the agent do exactly what you want? |
| 4 | **Pruning** | How do you make the skill as small as possible? |
